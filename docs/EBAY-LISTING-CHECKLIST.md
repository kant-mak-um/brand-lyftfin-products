# eBay Listing Checklist — LyftFin (R01)

This checklist standardizes eBay listings so they match the repo SKUs and reduce returns/confusion.

Source of truth:
- `docs/SKU-MATRIX.md`
- `catalog/products.json`
- `catalog/accessories.json`

---

## 0) One-minute “What am I listing?”
Choose exactly ONE listing type:

A) Surf/SUP fin blade (Dolphin / Shark / Sailfish)  
B) Modular mount (FCS II or Futures, cant 0/10/18)  
C) Bundle (single / thruster 3 / 5-fin)  
D) Longboard/SUP fin (Dolphin only, S/M/L × Hard/Flexible, integrated base)

If you can’t say which of the four it is, stop and re-check the SKU matrix.

## 2) Listing Title Template (copy/paste)
Use this structure so buyers immediately see compatibility:

### A) Surf/SUP fin blade
**LyftFin [Dolphin|Shark|Sailfish] Surf/SUP Fin Blade — Carbon Composite (Ixoskin/Isogrid) — Mount Sold Separately — [SKU]**

Example:
LyftFin Dolphin Surf/SUP Fin Blade — Carbon Composite (Ixoskin/Isogrid) — Mount Sold Separately — LF-FIN-DOL-STD-R01

### B) Modular mount
**LyftFin Mount — [FCS II|Futures] Compatible — [0°|10°|18° Cant] — Fits LyftFin Fins — [SKU]**

Example:
LyftFin Mount — FCS II Compatible — 10° Cant — Fits LyftFin Fins — LF-MNT-FCS2-C10-R01

### C) Bundle
**LyftFin [Single|Thruster 3|5-Fin] Set — [Dolphin|Shark|Sailfish] — Mounts Sold Separately — [SKU]**

LyftFin Thruster 3 Set — Shark — Mounts Sold Separately — LF-SET-3-SHK-R01

### D) Longboard/SUP fin
**LyftFin Longboard/SUP Dolphin Fin — [Small|Medium|Large] — [Hard|Flexible] — Integrated Base — [SKU]**

Example:
LyftFin Longboard/SUP Dolphin Fin — Medium — Flexible — Integrated Base — LF-LB-DOL-M-FLX-R01

## 3) Photos Checklist (minimum set)
**Required photos (all listings)**
- 1) Front full view
- 2) Side profile
- 3) Base / mounting interface close-up
- 4) Thickness/edge detail close-up (show stiffness build quality)
- 5) SKU label card in frame (handwritten is fine): “LyftFin — [SKU]”

**Additional for mounts**
- Show cant angle label: “0° / 10° / 18°”
- Show that mount fits LyftFin fin interface (close-up)

**Additional for longboard fins**
- Show integrated base clearly
- Show size comparison (S/M/L) if possible

## 5) Compatibility Warnings (must include)
Include the relevant warning lines in every listing:

- **FCS II vs Futures:** Buyer must match the fin system installed in their board (FCS II or Futures).
- **Mount sold separately** (for surf/SUP blades and bundles).
- **Longboard fins are integrated-base** (not modular).
- **Cant selection matters:** 0° / 10° / 18° are different.
- **Conversion is bolt-on:** not a quick swap.

**Trademark note (footer line):**
“FCS II and Futures are trademarks of their respective owners. LyftFin is not affiliated with or endorsed by them.”

## 7) Shipping & packaging checklist (minimum)
- Protect leading/trailing edges with foam/card
- Rigid mailer/box (no soft envelopes)
- Include printed SKU slip: “LyftFin — [SKU]”
- Include quick note: “Mount conversion is bolt-on; choose correct board system.”

---

## 8) Listing sanity checks (do not skip)
- Title contains: Brand + Type + Compatibility + Cant (if mount) + SKU
- Photos show: product + base/interface + SKU card
- Description states: what’s included + what’s not included
- Correct category chosen (fin vs mount vs longboard)
- SKU matches the matrix exactly (no typos)

---

## 9) Recommended listing order (fastest early traction)
1) Modular mounts (FCS II 0°, 10°, 18°)  
2) Dolphin fin blade (single)  
3) Thruster bundle (Dolphin)  
4) Futures mounts  
5) Shark and Sailfish blades/bundles  
6) Longboard Dolphin S/M/L hard + flexible
